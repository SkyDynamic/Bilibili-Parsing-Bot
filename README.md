## 功能

- 自动/手动解析BV号

## 安装

- 使用源码

# 帮助  
/bilitool help -> 显示这条信息 (等效指令:/帮助)  
/video + BV号 -> 解析视频  
/bilitool off -> 关闭自动解析  
/bilitool on -> 开启自动解析  
注：  
当自动解析开启后，会自动解析信息中第一个BV号的视频
